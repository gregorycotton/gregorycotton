From a number of videos taken on my friends' various digital cameras, my favourites are selected. A script scrapes YouTube for all videos with the same file name as those which were chosen. The downloaded videos are then arranged into movies, where instead of shots being connected by means of narrative, they are instead connected by the simple fact that they share a name. The result is an abstract film motivated by metadata, rather than plot.

One example has been shared on YouTube.
